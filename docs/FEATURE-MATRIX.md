# Feature matrix (free vs pro)

Draft — refine as modules land.

| Feature | Free | Pro / Edu |
|---------|------|-----------|
| Dark UI + 3D viewport | ✅ | ✅ |
| Drag-and-drop / palette part editor | ✅ | ✅ |
| Model-rocket class components | ✅ | ✅ |
| Basic trajectory sim + charts | ✅ | ✅ |
| Manual motor / mass / Cd entry | ✅ | ✅ |
| Report export (CSV) | ✅ optional | ✅ |
| Report export (PDF, full steps) | summary | full |
| Project cloud save | limited | unlimited |
| High-power / sounding models | ❌ | ✅ |
| Advanced numerics (DE, integration packs…) | ❌ | ✅ |
| Optimization / Monte Carlo | ❌ | ✅ |
| Sensor ingest + classification | ❌ | ✅ |
| Desktop app offline pro packs | free core | pro unlock |
| Team seats (later) | ❌ | planned |
| Admin console | ❌ (operators only) | ops |
